# UX 设计师作品集

> 来源文档：[作品集梳理](https://km.sankuai.com/collabpage/2754298145)

## 作品集策略

### 方向1（主打项目）— B端复杂系统设计

从仓储物流项目中挑选最能体现**系统复杂度 + 设计决策过程**的案例。

重点展示：**信息架构梳理、多角色流程拆解**、组件/规范体系建设，以及如何在业务约束下做设计取舍。

核心是展示怎么拆解复杂业务逻辑、怎么在多角色多场景中做设计决策，而不只是展示最终界面。

### 方向2（差异化项目）— 数据驱动 & 用户感知

主要是为了证明不只会做工具，也理解用户体验的"感知层"。

如果项目有数据验证（比如某个改版后转化率提升、用户留存变化）需要放进来。
